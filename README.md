# BrickAlpha

BrickAlpha is a LEGO market analytics platform that allows users to search LEGO sets, compare original retail prices with current market values, and visualize price trends like a stock dashboard.

## Phase 1 Features
- Search LEGO sets by set number
- Display set metadata (name, theme, year, pieces)
- Show retail vs current market price
- Calculate ROI
- Store price history
- Visualize price trends
- Homepage with featured sets and top market value sets

## Tech Stack
- Frontend: Next.js
- Backend: FastAPI
- Database: SQLite
